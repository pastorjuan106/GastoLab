💰 GastoLab

Aplicación web para la gestión de ingresos y gastos personales. Permite registrar movimientos, organizar categorías y visualizar la información mediante gráficos, brindando un control claro y sencillo de la situación financiera del usuario.

🚀 Demo en vivo

👉 https://gastolab-81bc2.web.app/

✨ Características
Registro e inicio de sesión de usuarios con google
Gestión de ingresos y gastos
Creación y administración de categorías
Creación y administración de movimientos
Visualización de datos mediante gráficos
Interfaz moderna, responsiva y fácil de usar
🧩 Tecnologías
React
Tailwind CSS
Supabase (Base de datos y autenticación)
Firebase Hosting (deploy)

⚙️ Instalación

Clona el repositorio y ejecuta el proyecto localmente:

git clone https://github.com/tu-usuario/gastolab.git
cd gastolab
npm install
npm run dev

🔐 Variables de entorno

Crea un archivo .env.local en la raíz del proyecto con las siguientes variables:

VITE_APP_SUPABASE_URL=#
VITE_APP_SUPABASE_ANON_KEY=# 
VITE_EMAILJS_PUBLIC_KEY=#
VITE_EMAILJS_SERVICE_ID=#
VITE_EMAILJS_TEMPLATE_ID=#
